# snippets
### Install Node.js

Download and install the node.js binaries for your platform from the [Node.js download page](http://nodejs.org/download/).

## Getting Started
```bash
# Install
npm install

# Start for development
npm run start:dev

# clean files
npm run clean

# Just run
npm run build
```
